# CGoLang Game of Life

A simulation of Conway's Game of Life built using Go and the Raylib graphics engine.

## Features

- **Real-time Dashboard:** Displays real-time metrics including live population count, simulation generations, and frame-rate diagnostics.
- **Interactive Canvas:** Fully responsive click-and-drag drawing mechanics to design or erase custom patterns on the fly.

## Controls

Launch the simulation to watch the classic **Gosper Glider Gun** cycle, or hit `Spacebar` to pause the simulation and sketch out cellular automata structures.

| Input                      | Action                               |
| :------------------------- | :----------------------------------- |
| `[Space]`                  | Pause / Unpause the simulation       |
| `[G]`                      | Toggle grid lines visibility         |
| `Left Mouse Click / Drag`  | Draw active cells (Only when paused) |
| `Right Mouse Click / Drag` | Erase cells (Only when paused)       |
